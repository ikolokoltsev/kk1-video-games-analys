# Video Game Genre Trends (KK1)

Analysis of how video game genre popularity has shifted from 1980 to 2024,
using global and regional sales data.

## Dataset

**Video Game Sales 2024** by asaniczka  
Source: https://www.kaggle.com/datasets/asaniczka/video-game-sales-2024

Download the dataset using the Kaggle CLI:

```bash
kaggle datasets download asaniczka/video-game-sales-2024 -p data/ --unzip
```

Install packages before downloading dataset.

## Packages install

```bash
pip install -r requirements.txt
```
